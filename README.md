# 🔁 C Recursion Practice — Functions & Algorithms

This repository contains recursion-based exercises in C programming.
The focus is on understanding how problems can be solved without loops, using recursive calls.

---

## 📚 Topics Covered

- Recursive functions
- Arrays with recursion
- Strings (recursive processing)
- Mathematical recursion
- Binary representation
- Searching in arrays/strings

---

## 📁 Tasks Overview

### 🟢 Task 1 — Fibonacci Number
Recursive calculation of the n-th Fibonacci number.

---

### 🟢 Task 2 — Product of Digits
Returns multiplication of all digits of a number recursively.

---

### 🟢 Task 3 — Print n → 0
Prints numbers from n down to 0.

---

### 🟢 Task 4 — Print 0 → n
Prints numbers from 0 up to n.

---

### 🟢 Task 5 — Print Array (Forward)
Recursively prints array elements in correct order.

---

### 🟢 Task 6 — Print Array (Reverse)
Recursively prints array elements in reverse order.

---

### 🟢 Task 7 — Sum of Digits
Calculates sum of digits using recursion.

---

### 🟢 Task 8 — Binary Representation
Prints a number in binary format using recursion.

---

### 🟡 Task 9 — Min Element in Array
Finds the smallest element in an array recursively.

---

### 🟡 Task 10 — Custom strlen
Recursive implementation of string length function.

---

### 🟡 Task 11 — First Occurrence of Character
Finds index of first occurrence of a character in a string.

---

### 🟡 Task 12 — First Uppercase Letter
Finds first uppercase letter in a string recursively.

---

## 🧠 Key Concepts Learned

- Divide & conquer thinking
- Stack-based recursion flow
- Base case importance
- Array traversal without loops
- String recursion handling
- Mathematical recursion patterns
